# isubo-init-conf
A cli-prompt for initializing isubo's configuration


# Usage

```shell
npx @isubo-org/init-conf

# ~ or ~
pnpm add @isubo-org/init-conf
npx init-conf

# ~ or ~
pnpm add @isubo-org/init-conf
node ./node_modules/.bin/init-conf
```
